# LocalFix

LocalFix is a hyperlocal service platform connecting users with nearby workers like electricians, plumbers, etc.

## Project Structure

This is a MERN stack project starter template.

* `client/`: React frontend (Vite + Tailwind CSS)
* `server/`: Node.js backend (Express + MongoDB)

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- MongoDB

### Backend Setup
1. Navigate to the server folder: `cd server`
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` in the root and fill in your details.
4. Start the server: `npm run dev`

### Frontend Setup
1. Navigate to the client folder: `cd client`
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
